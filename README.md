# Hacker News

Проект представляет собой fullstack приложение для чтения контента с Hacker News:

- **[hacker-news-backend](https://github.com/al-revenko/hacker-news-backend)** — NestJS REST API работающий с Firebase Hacker News API.
- **[hacker-news-frontend](https://github.com/al-revenko/hacker-news-frontend)** — Next.js веб-интерфейс.

## Как запустить

```bash
git clone https://github.com/al-revenko/hacker-news.git --recurse-submodules 

cd hacker-news

docker compose up
```
Сервисы доступны:
- **Frontend**:   http://localhost:3000
- **Backend**:    http://localhost:4000
